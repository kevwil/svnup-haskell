svnup-haskell

This is an archive of a little test code I wrote, to learn Haskell.

I blogged about it years ago, but that blog has been taken down. This archive is to preserve that code.

It's quite simple. Given a folder, this will loop through all direct child folders and will pull updates if a child folder is a Subversion working copy.
